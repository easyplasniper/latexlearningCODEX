# AGENTS.md

## Project goal
Build a beginner-friendly LaTeX tutorial project in Chinese for hands-on learning.

## Audience
Absolute beginners who do not know LaTeX well.
They learn best by editing small runnable examples and recompiling often.

## Teaching philosophy
- Prefer hands-on learning over long theory.
- Each lesson must be short, practical, and self-contained.
- Every lesson must include:
  1. learning goal
  2. runnable minimal example
  3. line-by-line explanation in simple Chinese
  4. 2-4 small edit tasks
  5. one common error section
- Explanations must avoid unnecessary jargon.
- When jargon is unavoidable, explain it in plain Chinese immediately.

## Project requirements
- Use Chinese as the main teaching language.
- Keep examples minimal and readable.
- Add abundant Chinese comments inside LaTeX source files.
- Prefer article class unless there is a clear reason otherwise.
- Ensure all .tex examples compile successfully.
- Use UTF-8 and beginner-safe packages.
- Avoid overly advanced macro programming.
- Avoid unnecessary visual complexity.

## Lesson scope
The course should cover:
1. first compilable document
2. document structure
3. sectioning and text formatting
4. math basics
5. figures
6. tables
7. references and bibliography
8. a final mini report project

## Output style
- Create clean folder structure.
- Write concise README with usage instructions for Overleaf and local LaTeX.
- Use progressive difficulty.
- Prefer small files over one huge file.
- Add “what to modify and observe” prompts in every lesson.

## Quality bar
Before finishing:
- verify file names and imports
- verify lesson order is progressive
- verify all examples are beginner-appropriate
- verify no broken references
- verify README matches actual files
